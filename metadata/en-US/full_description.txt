Features:
* Small size (<1MB)
* Adjustable keyboard height for more screen space
* Number row
* Swipe space to move pointer
* Delete swipe
* Custom theme colors
* Minimal permissions (only Vibrate)
* Ads-free

Feature it doesn't have and probably will never have:
* Emojis
* GIFs
* Spell checker
* Swipe typing

To enable the keyboard:
* Open "Hieroglyphs Keyboard" from your launcher
* Enable the Hieroglyphs Keyboard (default system warning about tracking will be shown)
* Switch to Hieroglyphs Keyboard from current Input Method (differs between keyboards, usually long-press space)
* To edit Hieroglyphs Keyboard settings long-press "," or open system Settings, Languages & Input, Hieroglyphs Keyboard.
